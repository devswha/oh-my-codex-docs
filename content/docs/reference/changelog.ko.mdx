## 최신 릴리스 — v0.13.1

detached interactive Codex startup을 복구하는 hotfix입니다.

**수정 사항:**
- detached tmux startup이 Codex stdin을 잃지 않습니다. detached leader wrapper가 backgrounding 과정에서도 stdin을 유지하므로, `omx --madmax --high` 같은 interactive detached launch가 시작 직후 종료되지 않습니다.
- CLI regression test에서 Codex child의 stdin이 열려 있는지, leader cleanup semantics가 보존되는지 검증합니다.
