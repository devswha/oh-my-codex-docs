## 最新版本 — v0.13.1

修复 detached interactive Codex 启动的 hotfix。

**修复内容：**
- detached tmux startup 不再丢失 Codex 的 stdin。detached leader wrapper 在后台化过程中保持 stdin，因此 `omx --madmax --high` 等 interactive detached launch 不再在启动后立即退出。
- CLI 回归测试验证 Codex child 的 stdin 是否打开，以及 leader cleanup semantics 是否保留。
