## 最新リリース — v0.13.1

detached interactive Codex startup を復元する hotfix です。

**修正内容:**
- detached tmux startup が Codex の stdin を失わなくなりました。detached leader wrapper がバックグラウンド化の過程でも stdin を保持するため、`omx --madmax --high` のような interactive detached launch が起動直後に終了しなくなります。
- CLI リグレッションテストで Codex child の stdin が開いているか、leader cleanup semantics が保持されているかを検証します。
