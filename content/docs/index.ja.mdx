**OpenAI Codex CLI 向けマルチエージェントオーケストレーション。学習コストゼロ。**

_Codex プロンプトを学ぶ必要はありません。OMX をそのまま使ってください。_

Oh My CodeX (OMX) は OpenAI Codex CLI の上で動作するワークフローレイヤーです。1 回の呼び出しで専用エージェントプロンプトを連携させ、複雑な開発タスクをエンドツーエンドで自動化します。

## クイックスタート

### 第 1 ステップ — Codex CLI をインストール

```bash
npm i -g @openai/codex
```

### 第 2 ステップ — OMX をインストール

```bash
npm i -g oh-my-codex
```

### 第 3 ステップ — セットアップ

```bash
omx setup
```

### 第 4 ステップ — OMX を起動して `$autopilot` を試す

まず OMX セッションを開始します:

```bash
omx
```

次に、そのセッション内で以下を入力します:

```
$autopilot build me a todo app in next.js
```

暗黙のキーワードトリガーを使う場合は、`autopilot build me a todo app in next.js` のように `$` なしでも動作します。
